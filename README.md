# Isibuwa Festival 2025 — Frontend

React + Vite + Tailwind CSS frontend for the Isibuwa Music Event Booking application.

## Quick Start

```bash
cp .env.example .env
# Set VITE_API_URL to your backend URL

npm install
npm run dev
```

See the [backend README](../Isibuwa_BE/README.md) and [DEPLOYMENT.md](../Isibuwa_BE/DEPLOYMENT.md) for full setup instructions.
